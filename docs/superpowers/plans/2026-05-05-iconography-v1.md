# Iconography v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the dual icon mechanism (per-icon Dioxus components + JS-loaded sprite) with a single sprite-only model: `web/build.rs` extracts SVGs from `web/assets/icons/{ui,narrative}/`, emits a sprite + a typed `IconName` enum + per-icon wrapper components; the sprite is inlined into the DOM at app root for first-paint availability.

**Architecture:** Source-of-truth lives in plain `.svg` files on disk (one per icon, `fill="currentColor"`). `build.rs` walks the icon folders and produces (a) `web/src/icons_generated.rs` containing `IconName`, `SPRITE`, and an `icon!` macro that defines per-icon wrapper components, plus (b) the sprite string itself. `Icon` primitive in `web/src/components/ui/icon.rs` renders `<svg><use href="#tier-name"/></svg>`. `App` mounts the sprite once via `dangerous_inner_html` at the root.

**Tech Stack:** Rust 1.95, Dioxus 0.7 (web/WASM), Tailwind CSS v4. The existing `web/build.rs` already generates a sprite from `web/src/components/icons/`; this plan rewires it to read from `web/assets/icons/` and emit Rust source instead of an asset file.

```rust
use dioxus::prelude::*;

/// Tier of an icon. Determines default size and signals visual register.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum IconTier {
    /// Geometric, Heroicons-style. 24x24 source viewBox.
    Ui,
    /// Illustrative, game-icons.net-style. 512x512 source viewBox.
    Narrative,
}

/// Standard icon size token. Maps to a Tailwind `size-*` utility.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum IconSize {
    Xs,
    Sm,
    Md,
    Lg,
    Xl,
    Xxl,
}

impl IconSize {
    /// Tailwind utility class to apply to the rendered `<svg>`.
    pub fn class(self) -> &'static str {
        match self {
            IconSize::Xs => "size-3",
            IconSize::Sm => "size-4",
            IconSize::Md => "size-5",
            IconSize::Lg => "size-8",
            IconSize::Xl => "size-12",
            IconSize::Xxl => "size-20",
        }
    }
}

impl IconTier {
    /// Default size token for this tier.
    pub fn default_size(self) -> IconSize {
        match self {
            IconTier::Ui => IconSize::Sm,
            IconTier::Narrative => IconSize::Lg,
        }
    }
}

/// Render an icon by referencing the inlined sprite via `<use>`.
///
/// Color comes from `currentColor` (i.e. parent `text-*` utility classes).
/// Pass an `aria_label` to make the icon meaningful; otherwise it is decorative
/// (`aria-hidden="true"`).
#[component]
pub fn Icon(
    /// Sprite ID to reference (e.g. `"ui-edit"` or `"narrative-fist"`).
    /// Use the codegen wrappers (`EditIcon`, `FistIcon`, ...) instead of
    /// constructing this primitive directly when the icon name is static.
    sprite_id: String,
    /// Source viewBox of the symbol. UI icons are `"0 0 24 24"`; narrative
    /// icons are `"0 0 512 512"`.
    view_box: String,
    /// Size token; pass `None` to use the wrapper's default.
    #[props(default)]
    size: Option<IconSize>,
    tier: IconTier,
    #[props(default)]
    class: String,
    #[props(default)]
    aria_label: Option<String>,
) -> Element {
    let size_class = size.unwrap_or_else(|| tier.default_size()).class();
    let combined_class = if class.is_empty() {
        size_class.to_string()
    } else {
        format!("{size_class} {class}")
    };
    let href = format!("#{sprite_id}");

    rsx! {
        if let Some(label) = aria_label {
            svg {
                class: "{combined_class}",
                view_box: "{view_box}",
                role: "img",
                aria_label: "{label}",
                focusable: "false",
                r#use { href: "{href}" }
            }
        } else {
            svg {
                class: "{combined_class}",
                view_box: "{view_box}",
                aria_hidden: "true",
                focusable: "false",
                r#use { href: "{href}" }
            }
        }
    }
}
```

fn main() {
    generate_env_file();

    println!("cargo:rerun-if-changed=assets/icons");

fn generate_icons_module() {
    let base = std::env::current_dir().expect("CARGO build cwd");
    let icons_root = base.join("assets/icons");
    let dest = base.join("src/icons_generated.rs");

    let mut entries: Vec<IconEntry> = Vec::new();
    collect_tier(&icons_root.join("ui"), "Ui", "0 0 24 24", &mut entries);
    collect_tier(
        &icons_root.join("narrative"),
        "Narrative",
        "0 0 512 512",
        &mut entries,
    );

    entries.sort_by(|a, b| a.variant.cmp(&b.variant));

    let mut out = String::new();
    out.push_str("// AUTO-GENERATED by build.rs. Do not edit by hand.\n");
    out.push_str("// Edit SVGs in web/assets/icons/{ui,narrative}/ and rebuild.\n\n");

    out.push_str("use dioxus::prelude::*;\n");
    out.push_str("use crate::components::ui::{Icon, IconSize, IconTier};\n\n");

    // Inline sprite string.
    out.push_str("pub const SPRITE: &str = \"<svg xmlns=\\\"http://www.w3.org/2000/svg\\\" style=\\\"display:none\\\">");
    for e in &entries {
        out.push_str(&format!(
            "<symbol id=\\\"{}\\\" viewBox=\\\"{}\\\">{}</symbol>",
            escape_for_rust_str(&e.sprite_id),
            escape_for_rust_str(&e.view_box),
            escape_for_rust_str(&e.body),
        ));
    }
    out.push_str("</svg>\";\n\n");

    // IconName enum.
    out.push_str("#[derive(Debug, Clone, Copy, PartialEq, Eq)]\n");
    out.push_str("pub enum IconName {\n");
    for e in &entries {
        out.push_str(&format!("    {},\n", e.variant));
    }
    out.push_str("}\n\n");

    out.push_str("impl IconName {\n");
    out.push_str("    pub fn sprite_id(&self) -> &'static str {\n");
    out.push_str("        match self {\n");
    for e in &entries {
        out.push_str(&format!(
            "            IconName::{} => \"{}\",\n",
            e.variant, e.sprite_id
        ));
    }
    out.push_str("        }\n    }\n\n");

    out.push_str("    pub fn view_box(&self) -> &'static str {\n");
    out.push_str("        match self {\n");
    for e in &entries {
        out.push_str(&format!(
            "            IconName::{} => \"{}\",\n",
            e.variant, e.view_box
        ));
    }
    out.push_str("        }\n    }\n\n");

    out.push_str("    pub fn tier(&self) -> IconTier {\n");
    out.push_str("        match self {\n");
    for e in &entries {
        out.push_str(&format!(
            "            IconName::{} => IconTier::{},\n",
            e.variant, e.tier
        ));
    }
    out.push_str("        }\n    }\n");
    out.push_str("}\n\n");

    // Per-icon wrapper components.
    for e in &entries {
        out.push_str(&format!(
            "#[component]\n\
            pub fn {wrapper}(\n    \
                #[props(default)] size: Option<IconSize>,\n    \
                #[props(default)] class: String,\n    \
                #[props(default)] aria_label: Option<String>,\n\
            ) -> Element {{\n    \
                rsx! {{\n        \
                    Icon {{\n            \
                        sprite_id: \"{sprite_id}\".to_string(),\n            \
                        view_box: \"{view_box}\".to_string(),\n            \
                        tier: IconTier::{tier},\n            \
                        size, class, aria_label\n        \
                    }}\n    \
                }}\n\
            }}\n\n",
            wrapper = e.wrapper,
            sprite_id = e.sprite_id,
            view_box = e.view_box,
            tier = e.tier,
        ));
    }

    std::fs::write(&dest, out).expect("write icons_generated.rs");
}

struct IconEntry {
    /// snake_case file stem, e.g. "eye_open"
    name: String,
    /// CamelCase enum variant, e.g. "EyeOpen"
    variant: String,
    /// CamelCase wrapper component, e.g. "EyeOpenIcon"
    wrapper: String,
    /// Sprite ID, e.g. "ui-eye-open"
    sprite_id: String,
    /// Source viewBox, e.g. "0 0 24 24"
    view_box: String,
    /// Tier name as Rust enum variant, "Ui" or "Narrative"
    tier: &'static str,
    /// Inner SVG body (everything between <svg ...> and </svg>).
    body: String,
}

fn collect_tier(dir: &std::path::Path, tier: &'static str, view_box: &str, out: &mut Vec<IconEntry>) {
    if !dir.is_dir() {
        return;
    }
    let entries = std::fs::read_dir(dir).expect("read icons tier dir");
    for entry in entries.flatten() {
        let path = entry.path();
        if path.extension().and_then(|s| s.to_str()) != Some("svg") {
            continue;
        }
        let stem = path.file_stem().and_then(|s| s.to_str()).expect("stem").to_string();
        let raw = std::fs::read_to_string(&path).expect("read svg");
        let body = extract_svg_body(&raw);
        let prefix = match tier {
            "Ui" => "ui-",
            "Narrative" => "narrative-",
            _ => unreachable!(),
        };
        let sprite_id = format!("{}{}", prefix, stem.replace('_', "-"));
        let variant = snake_to_camel(&stem);
        let wrapper = format!("{}Icon", variant);
        out.push(IconEntry {
            name: stem,
            variant,
            wrapper,
            sprite_id,
            view_box: view_box.to_string(),
            tier,
            body,
        });
    }
}

/// Take everything between the opening `<svg ...>` tag and the closing `</svg>`.
/// Drops the outer `<svg>` so we can wrap with our own `<symbol>`.
fn extract_svg_body(raw: &str) -> String {
    let after_open = match raw.find('>') {
        Some(i) => &raw[i + 1..],
        None => return String::new(),
    };
    let before_close = match after_open.rfind("</svg>") {
        Some(i) => &after_open[..i],
        None => after_open,
    };
    before_close.trim().to_string()
}

fn snake_to_camel(s: &str) -> String {
    let mut out = String::new();
    let mut upper = true;
    for ch in s.chars() {
        if ch == '_' {
            upper = true;
        } else if upper {
            out.extend(ch.to_uppercase());
            upper = false;
        } else {
            out.push(ch);
        }
    }
    out
}

fn escape_for_rust_str(s: &str) -> String {
    s.replace('\\', "\\\\").replace('"', "\\\"")
}
